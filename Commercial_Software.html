Prelight, FilmLight Look creation software for Mac OS based around BLG (Baselight Saved Grade) but also capable of working in a more limited ASC CDL mode. Available in a free version which allows a DIT or Cinematographer to work with the full Baselight toolset to create looks using still images, and a paid version which can control LUT boxes for live grading and metadata management. https://www.filmlight.ltd.uk/products/prelight/overview_pl.php
